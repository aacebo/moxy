# Moxy

Derive macros that eliminate Rust boilerplate. Get `Display`, `Deref`, `Default`, and `Build` implementations with a single attribute.

## Install

```toml
[dependencies]
moxy = { version = "0.0.0", features = ["derive"] }
```
